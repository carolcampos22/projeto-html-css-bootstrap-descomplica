# Projeto Campos Rock Shop!

## Objetivos
Criação de uma loja virtual de produtos voltados ao público roqueiro. Os produtos são divididos em quatro categorias:
 - roupas
 - sapatos
 - instrumentos musicais
 - acessórios

 ## Tecnologias utilizadas
 - HTML
 - CSS
 - Bootstrap
